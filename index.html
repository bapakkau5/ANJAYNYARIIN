Android ‚Äî Powertunnel
Klik untuk buka
  1. Unduh aplikasi Powertunnel pada Android.
  2. Kembali ke halaman ini lalu klik tombol Copy Powertunnel (Android) di atas untuk menyalin URL host.
  3. Buka Powertunnel ‚Üí ikon tiga garis (menu) ‚Üí Host Settings.
  4. Paste URL host yang sudah disalin ke kolom host.
  5. Ubah host update period dari 12 jam ‚Üí on start.
  6. Aktifkan Powertunnel lalu buka Growtopia dan login.
iOS ‚Äî Surge 5
Klik untuk buka
  1. Install aplikasi Surge 5 dari App Store.
  2. Buka Surge ‚Üí tekan OK ‚Üí pilih Default.conf.
  3. Pilih IMPORT ‚Üí Download Profile from URL.
  4. Kembali ke halaman ini lalu klik Copy iOS (Surge 5) untuk menyalin URL konfigurasi.
  5. Paste URL di Surge ‚Üí OK ‚Üí Done ‚Üí Setup ‚Üí izinkan konfigurasi VPN.
  6. Buka Growtopia dan Connect.

Profile URL contoh: https://bapakkau5.github.io/GTRGIOS/ios.txt

Windows ‚Äî Edit hosts
Klik untuk buka
  1. Tekan Win + R, ketik: C:\Windows\System32\drivers\etc lalu Enter.
  2. Buka file hosts dengan Notepad (run as admin bila perlu).
  3. Tambahkan baris IP di bawah ini pada akhir file lalu simpan:
