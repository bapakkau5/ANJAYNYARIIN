How to Play (Panduan)

  1. Unduh aplikasi Powertunnel di perangkat Android.
  2. Tekan Copy Powertunnel (Android) untuk menyalin URL host.
  3. Buka Powertunnel → Menu → Host Settings → tempel URL.
  4. Ubah host update period menjadi on start.
  5. Aktifkan Powertunnel lalu buka Growtopia dan login.
